# Success Metrics Frameworks

## When to Use This Skill

- Defining KPIs
- Measuring product success
- Creating dashboards

## Core Concepts

### HEART Framework (Google)

| Metric | Description | Example |
|--------|-------------|---------|
| **Happiness** | User satisfaction | NPS, CSAT |
| **Engagement** | User involvement | DAU, session length |
| **Adoption** | New users | Sign-ups, activations |
| **Retention** | Returning users | Day 1/7/30 retention |
| **Task Success** | Goal completion | Conversion rate |

### AARRR (Pirate Metrics)

- **Acquisition**: How users find you
- **Activation**: First positive experience
- **Retention**: Users coming back
- **Revenue**: Monetization
- **Referral**: Users recommending

### Metric Types

**North Star Metric:**
- Single metric that best captures value
- Example: "Weekly Active Users"

**KPIs:**
- Measurable values indicating success
- Example: "Conversion rate", "Retention rate"

**Leading Indicators:**
- Predict future performance
- Example: "Sign-up rate"

**Lagging Indicators:**
- Measure past performance
- Example: "Revenue", "Churn rate"

## Setting Targets

- Define baseline
- Set realistic targets
- Review regularly
